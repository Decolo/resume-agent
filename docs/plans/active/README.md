# Active Plans

This folder is the source of truth for plans currently in execution.

## Usage

1. Place one file per active initiative.
2. Keep status explicit (`planned`, `in_progress`, `blocked`, `done`).
3. Link PRs/issues for each milestone.
4. Move completed plans to `docs/plans/completed/`.

## Suggested Template

```md
# <Plan Title>

Status: in_progress
Owner: <name>
Start: YYYY-MM-DD
Target: YYYY-MM-DD

## Goal

## Milestones

## Risks

## Validation
```
